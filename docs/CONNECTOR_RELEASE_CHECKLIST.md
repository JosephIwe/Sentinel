# Connector Release Checklist

Use this checklist when adding a new connector to Sentinel, one at a time.

- [ ] Feature branch created
- [ ] Connector implemented
- [ ] SUCCESS/NO_DATA/ERROR verified
- [ ] Diagnostics included
- [ ] Evidence objects validated
- [ ] Unit tests added
- [ ] Integration tests added
- [ ] npm test passes
- [ ] npm run build passes
- [ ] Real-domain smoke tests run
- [ ] README updated
- [ ] CHANGELOG updated
- [ ] Connector scorecard updated
- [ ] PR created
- [ ] Squash merged
- [ ] Alpha tag created
